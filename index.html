Ciljevi projekta

Kroz ovaj projekt primjenjujem stečeno znanje iz HTML-a i CSS-a te stvoriti funkcionalnu i vizualno privlačnu web stranicu (nadam se da ce ispast dobro na kraju hahaha).

Stvari koje moram iskoristiti i napraviti:

    • Naučiti složene CSS selektore
    • Koristiti display, float, list-style
    • Poboljšati tipografiju sa text-align

Zanimljivosti

CSS je izašao 1996. godine, a HTML 1993.
