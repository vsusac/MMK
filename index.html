Ciljevi projekta

Kroz ovaj projekt primjenjujem stečeno znanje iz HTML-a i CSS-a te stvoriti funkcionalnu i vizualno privlačnu web stranicu (nadam se da će ispasti dobro na kraju hahaha).

Stvari koje moram iskoristiti i napraviti:

  1. Naučiti složene CSS selektore
  2. Koristiti display, float, list-style
  3. Poboljšati tipografiju sa text-align

Zanimljivosti

Značenje HTML-a je HyperText Markup Language, a značenje CSS-a je Cascading Style Sheets.

CSS je izašao 1996. godine, a HTML 1993.
